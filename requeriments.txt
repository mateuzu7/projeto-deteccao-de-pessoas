# Por que esses itens?

torch, torchvision e torchaudio: São a base para rodar o modelo Mask R-CNN.
opencv-python: Necessário para manipular as imagens e rodar a webcam.
numpy: Essencial para os cálculos das matrizes das imagens.
matplotlib: Usado pelo Detectron2 para desenhar as previsões nas fotos.
pyyaml: O Detectron2 usa arquivos .yaml para carregar as configurações do modelo
